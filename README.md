# 🎬 MovieMate

MovieMate is an AI-powered movie recommendation system that provides personalized movie recommendations based on movie name, mood, cast, and text input using Machine Learning and Natural Language Processing techniques.

## Features

- 🎥 Movie-based recommendations
- 😊 Mood-based recommendations
- 🎭 Cast-based recommendations
- ✍️ Text-based recommendations
- 🎤 Voice input
- 🌙 Dark mode

## Tech Stack

- React
- Flask
- Python
- Pandas
- Scikit-learn
- Hugging Face Transformers
- RapidFuzz
- NLTK

## Run Locally

### Backend

```bash
cd BACKEND
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
